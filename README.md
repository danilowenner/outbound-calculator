# Markuo · Calculadora Outbound

Calculadora de proyección outbound para Cold Email, LinkedIn Outreach y Cold Calling.

## Desarrollo local

```bash
npm install
npm run dev
```

## Build de producción

```bash
npm run build
```

## Deploy

Este proyecto está optimizado para deployar en Vercel. Conectá el repo y Vercel detecta Vite automáticamente.
